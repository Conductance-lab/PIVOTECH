UF2 刷写指南
① 进入 Bootloader 模式
  • RP2040（Pico）：按住 BOOTSEL 键，再连接 USB
  • STM32：按住 BOOT0 键，再连接 USB 或上电
  • ESP32-S2/S3：按住 BOOT 键，再按 RST 键
② 复制固件

设备将显示为 USB 大容量存储设备（如 RPI-RP2）。将下载的 .uf2 文件直接拖放到该驱动器中。

③ 验证

刷写完成后设备将自动重启。返回"串口终端"页，选择正确波特率（默认 115200），点击"连接串口"，页面将自动发送握手命令验证设备。

项目简介

PIVOTECH 是一个针对嵌入式平台的通用快速调试检测器网页工具。用户无需安装任何软件，直接在浏览器中打开 GitHub Pages 静态网页即可使用。


通过 Web Serial API 与硬件进行通信，支持数据读取、参数控制和设备验证。

核心功能
  • ✓ Web Serial API 串口通信
  • ✓ 握手命令 + 响应验证，自动识别设备
  • ✓ 实时数据终端（ASCII / HEX 切换）
  • ✓ 参数读写控制面板
  • ✓ 历史 UF2 固件版本下载
  • ✓ 纯静态网页，无需后端
设备握手协议

连接串口后，网页自动发送握手命令（默认 PIVOTECH_ID\r\n），等待设备响应中包含预期标识符（默认 PIVOTECH）。


握手命令和期望响应均可在"串口终端"页自定义，以适配不同固件版本。

浏览器支持
  • ✅ Google Chrome 89+
  • ✅ Microsoft Edge 89+
  • ✅ Opera 75+
  • ❌ Firefox（不支持 Web Serial API）
  • ❌ Safari（不支持 Web Serial API）

需要在 HTTPS 或 localhost 环境下运行。
